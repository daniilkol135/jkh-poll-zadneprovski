# Опрос качества услуг ЖКХ

Сайт для сбора оценок качества услуг ЖКХ от жителей района.

## Как использовать:
1. Перейдите на сайт: [ВАШ_АДРЕС.netlify.app](https://ваш-адрес.netlify.app)
2. Заполните форму
3. Посмотрите статистику

## Техническая информация:
- Фронтенд: HTML/CSS/JavaScript
- Хостинг: Netlify + GitHub Pages
- Формы: Formspree
- Данные: Google Sheets
